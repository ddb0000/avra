## Setup and Running

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Configure Gemini API Key:**
   - Create a `.env` file in the project root
   - Add your Gemini API key:
     ```
     GEMINI_API_KEY=your_api_key_here
     ```

3. **Run the Application:**
   ```bash
   python gui.py
   ```
